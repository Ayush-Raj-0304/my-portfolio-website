## ‚ú® Key Features

- **Multi-page Blog Structure**: Implemented using React Router and Hooks, providing a seamless SPA experience with 30% faster page transitions.
- **Centralized State Management**: Simplified and optimized user authentication using Redux Toolkit, reducing complexity by 40%.
- **Reusable UI Components**: Created a library of modular components (buttons, forms, inputs) using React Hook Form, which cut down repetitive code by 25%.
- **Backend with Appwrite**: Handled authentication, database, and file storage using Appwrite's modular backend-as-a-service model. Reduced vendor lock-in by 30%.
- **Rich Text Editing**: Integrated TinyMCE HTML editor for a fully featured post-writing experience.
- **HTML Parsing**: Used HTML React Parser to safely render HTML content in the frontend.
